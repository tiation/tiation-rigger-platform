# MetricsDashboard

Enterprise-grade component for the RiggerConnect platform.

## Overview

This component is part of the RiggerConnect ecosystem, designed for professional workflow management and rigger job coordination.


## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/rigger-connect-MetricsDashboard.git
cd rigger-connect-MetricsDashboard

# Install dependencies
npm install

# Setup environment
cp .env.example .env
# Edit .env with your configuration
```

## Usage

### Development

```bash
# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```

### Production Deployment

```bash
# Build and deploy
npm run build
npm run deploy
```
